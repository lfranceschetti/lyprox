Lymphatic Progression eXplorer

Welcome to our online interface with which we aim to visualize the patterns of lymphatic progression in head and neck squamous cell carcinoma.

{% include_md 'index.md' %}

Features

Dashboard

Interactive page that aggregates information on all patient's lymphatic metastatic involvement in a visual and intuitive dashboard. Here, one can explore correlations between involvements of different lymph node levels as well as estimate the impact of various risk factors on nodal involvement.

List of patients

Look at a table of all the patients in the database. This view can be filtered for different patient-specific properties like age, date of diagnosis, TNM stage and more.

Export database *

Download a CSV table of all patients in the database. This table can also be used as a template for uploading a batch of patients, as it has the required format for uploading.

Create new patient *

Manually add a new patient to the database using a pre-defined form.

Upload patients *

Add a batch of patients to the database via a CSV table of a specific format.

* login required
